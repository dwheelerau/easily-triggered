Important configuration settings

Megadetector provides a probability of an animal detection. This can be used as a cut-off to avoid too many false positives or false negatives. It is worth optimising this value as the best value will depend on the data. The options are a range from 0.0 to 1.0. We recommend a value of 0.2-0.5, with 0.3 as the default. A value of 0.3 sets the detection cut-off at a probability (of an animal) of 30%.
